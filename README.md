# pepsico-sales-analytics-powerbi-dashboard
This Power BI dashboard project analyzes sales performance, distributor activity, and regional trends. It includes KPIs, monthly sales insights, category breakdown, and target comparisons. The dashboard helps identify revenue patterns, optimize distribution strategies, and support data-driven business decisions.
## 🎯 Objectives
- Analyze monthly sales trends  
- Evaluate distributor performance  
- Identify top-performing product categories  
- Compare sales targets with actual results  
- Provide actionable business insights  

---

## 🗂️ Dataset Description
The dataset consists of three main tables:

### 🔹 Sales
- Sales volume  
- Revenue  
- Product category  
- Month & Year  

### 🔹 Distributor
- Distributor ID  
- City  
- Distribution regions  

### 🔹 Shop
- Shop name & code  
- Regional Manager  
- Unit Manager  
- Order Taker  

---

## 📊 Dashboard Features

- 📈 **Sales by Month** – Tracks trends and seasonality  
- 🥧 **Sales by Category** – Shows product contribution  
- 📍 **Distributors by City** – Highlights regional performance  
- 🎯 **Sales vs Target** – KPI comparison  
- 🗺️ **Geographical Analysis** – Map-based insights  
- 👥 **Manager Performance** – Shop-level analysis

## 🔍 Key Insights
- Sales show fluctuations across months indicating seasonal trends  
- A few distributors dominate overall sales contribution  
- Plastic bottle category generates the highest revenue  
- Some regions underperform despite potential  
- Gap observed between target and actual sales
